Python Learning Repository

This repository is created for systematic learning of the Python programming language from beginner to a confident level. It contains practical examples, exercises, and small projects to help reinforce knowledge through hands-on practice.

Goal

Learn the basic Python syntax

Understand core programming concepts

Write clean and readable code

Gain practical experience through exercises and mini-projects

Prepare for more advanced areas (automation, backend development, data analysis)

How to Use

Move step by step from simple topics to more complex ones

Write code yourself — avoid copying without understanding

Practice after each topic

Revisit difficult concepts regularly

Add your own examples and notes

Tools

Python 3.x

Code editor (VS Code / PyCharm)

Git for version control

Learning Plan

Days 1–7: Basic syntax

Days 8–14: Conditions and loops

Days 15–21: Functions and data structures

Days 22–30: Working with files and modules

Month 2+: OOP and first projects

Rules

Keep code clean and readable

Separate topics into different files

Add comments where necessary

Commit changes regularly with clear messages

Progress

This repository will be updated gradually throughout the learning process.
